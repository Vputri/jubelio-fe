üìä DATABASE MODELS


class Category(models.Model):
    name = models.CharField(max_length=100)

class SubCategory(models.Model):
    name = models.CharField(max_length=100)
    category = models.ForeignKey(Category, on_delete=models.CASCADE)

class Product(models.Model):
    name = models.CharField(max_length=200)
    subcategory = models.ForeignKey(SubCategory, on_delete=models.CASCADE)
    unit_price = models.DecimalField(max_digits=10, decimal_places=2)

class Sales(models.Model):
    product = models.ForeignKey(Product, on_delete=models.CASCADE)
    quantity = models.IntegerField()
    discount = models.DecimalField(max_digits=3, decimal_places=2)
    profit = models.DecimalField(max_digits=10, decimal_places=2)
    state = models.CharField(max_length=50)
    region = models.CharField(max_length=50)
    date = models.DateField()

üìä DASHBOARD & FILTER

Visualisasi Data

  • Pie Chart - Most Sold Products: Menampilkan proporsi penjualan tiap produk. Irisan terbesar = produk terlaris.
  • Scatter Plot - Discount vs Quantity Sold: Menunjukkan hubungan antara diskon dan jumlah terjual. Setiap titik = wilayah, warna = kuantitas.
  • Heatmap - Product Sales by State: Distribusi penjualan berdasarkan negara/wilayah dan kategori. Warna gelap = penjualan tinggi.

Cara Penggunaan Filter

  • Gunakan dropdown Category, Subcategory, Country, State, City, Region untuk memfokuskan analisis.
  • Contoh: Filter "Technology" di "California" untuk melihat performa spesifik.
  • Klik Reset Filters untuk kembali ke seluruh data.

Insight & Manfaat

  • Identifikasi produk terlaris dan kontribusinya.
  • Analisis pengaruh diskon terhadap penjualan.
  • Temukan wilayah/kategori dengan performa terbaik atau peluang pengembangan.

üéØ MANFAAT BISNIS

Analytics Capabilities

  • Product Performance Analysis: Menganalisis performa setiap produk berdasarkan data penjualan.
    • Contoh: Pie Chart "Most Sold Products" menunjukkan produk terlaris. Filter dapat digunakan untuk analisis per kategori/region.
  • Pricing Optimization: Menentukan harga dan diskon optimal agar penjualan meningkat tanpa mengorbankan margin.
    • Contoh: Scatter Plot "Discount vs Quantity Sold" memperlihatkan pengaruh diskon terhadap penjualan. Bisa dianalisis diskon optimal untuk tiap kategori.
  • Geographic Insights: Mengetahui wilayah dengan penjualan tertinggi/rendah untuk strategi distribusi dan pemasaran.
    • Contoh: Heatmap "Product Sales by State" memperlihatkan penjualan per negara/kategori. Dapat ditemukan peluang ekspansi wilayah.
  • Trend Analysis: Menganalisis tren penjualan dari waktu ke waktu untuk prediksi permintaan dan deteksi penurunan performa.
    • Contoh: Jika dashboard menampilkan data per bulan/tahun, tren naik/turun bisa diidentifikasi untuk strategi bisnis ke depan.

Strategic Benefits

  • Data-Driven Decisions: Pengambilan keputusan bisnis berbasis data nyata, bukan asumsi, sehingga lebih akurat dan terukur.
    • Contoh: Manajemen menambah stok produk terlaris berdasarkan data pie chart, atau menghentikan promosi di wilayah yang tidak responsif terhadap diskon.
  • Market Expansion: Identifikasi wilayah/kategori potensial untuk ekspansi pasar berdasarkan data penjualan aktual.
    • Contoh: Dari heatmap, region "West" masih rendah penjualannya untuk kategori Furniture, sehingga bisa jadi target ekspansi.
  • Inventory Management: Optimasi stok barang dengan memantau tren penjualan dan menghindari overstock/stockout.
    • Contoh: Jika penjualan produk tertentu melonjak di satu wilayah, stok bisa segera ditambah. Produk stagnan bisa dikurangi stoknya.
  • Pricing Strategy: Menentukan strategi harga dan diskon yang optimal berdasarkan analisis korelasi diskon dan penjualan.
    • Contoh: Dari scatter plot, bisa diketahui diskon berapa persen yang paling efektif menaikkan penjualan untuk tiap kategori/wilayah.
