Prerequisites:

  • Node.js (version 16 atau lebih tinggi)
  • npm atau yarn
  • Python 3.8+ (untuk Django backend)
  • Django server running di port 8000

Frontend Setup:

cd jubelio-fe

# Install dependencies

# Start development server

Backend Setup:

# Lihat cors-config.md untuk detail
